# Andromeda Minecraft Server

Instructions on how to join the Andromeda Minecraft Server and install all required mods.

> **Minecraft Version 26.1.x is REQUIRED**

## Mod Installation

### Step 1 — Download and Install Fabric

Download the Fabric Installer from: <a href="https://fabricmc.net/use/installer/" target="_blank">https://fabricmc.net/use/installer/</a>

<img src="/img/Fabric%20Installer%20Download.webp" width="700" alt="Fabric website showing the Download for Windows button">

> Make sure Minecraft and the Minecraft Launcher are **closed** before proceeding.

Open the installer, confirm the Minecraft version is **26.1.x**, then click **Install**.

<img src="/img/Fabric%20Installer%20Installing.webp" width="700" alt="Fabric Installer dialog showing the Install button">

### Step 2 — Select the Fabric Profile in the Launcher

Open the Minecraft Launcher. You should see the Fabric profile appear in the profile list.

<img src="/img/Minecraft%20Fabric%20Selection.webp" width="700" alt="Minecraft Launcher showing the Fabric profile selected">

### Step 3 — Open the Minecraft Folder

Navigate to the **Installations** tab and click the folder icon next to the Fabric profile to open its installation folder.

<img src="/img/Navigate%20to%20Minecraft%20Folder.webp" width="700" alt="Minecraft Launcher Installations tab with the folder icon highlighted">

You should see a **`mods`** folder inside. If it does not exist, create a new folder and name it exactly **`mods`**.

<img src="/img/Mods%20Folder.webp" width="700" alt="Minecraft installation folder with the mods folder highlighted">

### Step 4 — Download and Install the Mods Pack

<a href="https://github.com/Nattaphong-jul/Andromeda-Minecraft-Server/releases/tag/1.7.9" target="_blank">&gt; Download Mods Pack here</a>

<img src="/img/Release%20mods%20pack.webp" width="700" alt="GitHub release page showing the Andromeda.Mods.zip download">

Extract the ZIP file and move all contents into the **`mods`** folder.

<img src="/img/Move%20mods%20to%20the%20folder.webp" width="700" alt="Moving extracted mod files into the mods folder">

Your `mods` folder should look like this:

<img src="/img/Mods%20folder%20example.webp" width="700" alt="Example of a correctly populated mods folder">

---

## Joining the Server

Mod installation is complete! Launch Minecraft using the **Fabric** profile, go to **Multiplayer**, and connect using the server address below:

```
andromeda.minecraft.party
```

## Shader Pack (Optional)

Shaders are not required but significantly improve the visual experience.

### Step 1 — Download the Shader Pack

<a href="https://github.com/Nattaphong-jul/Andromeda-Minecraft-Server/releases/tag/1.7.9" target="_blank">&gt; Download Shader Pack here</a>

<img src="/img/Shader%20Pack%20Download.webp" width="700" alt="GitHub release page showing the shader pack ZIP download">

### Step 2 — Create the `shaderpacks` Folder

In your Minecraft installation folder, create a new folder named exactly **`shaderpacks`**.

<img src="/img/Navigate%20to%20Minecraft%20Folder.webp" width="700" alt="Minecraft installation folder">

<img src="/img/Shader%20Pack%20Folder.webp" width="700" alt="New shaderpacks folder created in the Minecraft folder">

### Step 3 — Add the Shader ZIP

Move the downloaded ZIP file **directly** into the `shaderpacks` folder. **Do not extract it.**

<img src="/img/Shader%20ZIP.webp" width="700" alt="Shader ZIP file placed inside the shaderpacks folder">

### Step 4 — Enable the Shader In-Game

Open Minecraft and navigate to **Options → Video Settings** (not Resource Packs).

<img src="/img/Video%20Setting.webp" width="700" alt="Minecraft Video Settings menu">

You should see the **Shader Packs** option in the left menu.

<img src="/img/Iris%20Shader%20Pack.webp" width="700" alt="Shader Packs option visible in the Video Settings left menu">

Select the shader pack and click **Apply**.

<img src="/img/Select%20the%20shader%20pack.webp" width="700" alt="Selecting and applying the shader pack">

### All done — enjoy the server! :)
